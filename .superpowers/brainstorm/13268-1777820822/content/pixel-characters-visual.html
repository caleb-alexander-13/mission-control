Pixel Agent Characters

Which style feels right? Each agent is a person holding their domain object.

A

Simple Blocky (8-bit)

Basic rectangular body, stick arms, holding object prominently. Very retro, clean.

B

Detailed Pixel (16-bit)

More curves, defined limbs, head with simple face. Less retro, more character.

C

Mini Cute

Tiny characters with round heads. Cute and modern, works well in tight spaces.
